# JiaGuoMeng
家国梦建筑摆放计算

## 使用方法

Step1. 在文件开头输入你有的三类建筑、建筑的星级、照片、政策和任务加成

Step2. 用python3运行程序

## 说明

- 程序假设所有建筑等级相同**这一假设可能会造成和输出最优与实际最优不符**

- 部分高星建筑数据缺失，欢迎补充、修改
