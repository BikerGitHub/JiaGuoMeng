# JiaGuoMeng
家国梦建筑摆放计算

## 使用方法

### 参数输入的两种方式:
- 写入到data.txt文件内
- 写入到jiaguomeng.py开头JsonData={}中
### 参数说明
#### Mode
- "Online":计算在线模式加成
- "Offline":计算离线模式加成
#### LastResult
- 可将上次输出的建筑策略记录,下次计算会出现替换操作提示
#### Builds
- 将建筑按星级输入,以空格分割
#### 数值调整
数值调整均为增加的部分,即游戏内显示增加150%则输入1.5
##### Mission
- 以 "建筑":加成 的方式输入
- 使用"住宅","商业","工业","Online","Offline"来输入对某类情况的加成
- 注意大括号中最后一行末尾没有逗号
##### Policy&Photos
- "Online":在线加成
- "Offline":离线加成
- "Residence":住宅加成
- "Commercial":商业加成
- "Industry":工业加成
## 说明

- 游戏不同建筑起始收益略有区别，这里只收集到了*平房*和*民食斋*的数据

- 程序假设所有建筑等级相同**这一假设可能会造成和输出最优与实际最优不符**

- 部分高星建筑数据缺失，欢迎补充、修改
